# Hamza Express Website

## Brand-Compliant Professional Website
**Over 100 Years of Taste, Delivered · Est. 1918**

### Quick Setup for Antigravity

1. **Import the folder** into Antigravity
2. **All assets are already named correctly** (underscores, no spaces)
3. **Run preview**: `npx serve .`
4. **Open**: `http://localhost:3000`

### Asset Files in `/assets` Folder

| File Name | Purpose |
|-----------|---------|
| `Full_Logo_White_BG.svg` | Full logo on white background (hero, footer) |
| `Full_Logo_Tan_BG.svg` | Full logo on tan background (alternate use) |
| `Full_Logo_Brown_BG.svg` | Full logo on brown background (dark sections) |
| `Icon.svg` | Founder portrait icon (favicon, loader, legacy section) |
| `Icon.png` | PNG version of icon (Apple touch icon) |
| `Text_Logo.svg` | Text-only "Hamza EXPRESS" logo (navigation) |
| `Text_Logo.png` | PNG version of text logo |

### GitHub Deployment

```bash
git init
git add .
git commit -m "Initial commit: Hamza Express website"
git remote add origin https://github.com/YOUR_USERNAME/hamza-express-site.git
git push -u origin main
```

Then enable GitHub Pages or Cloudflare Pages.
