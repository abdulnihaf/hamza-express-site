Deployment commands

All 16 creatives are confirmed in the PiSignage library. Deploy with:

cd /Users/nihaf/Documents/Tech/HN-Hotels-Site/.claude/worktrees/optimistic-elion-2712b1
ENGINE=projects/pisignage-hamza-express/engine/ps_engine.py
SCENE=projects/pisignage-hamza-express/scenes/psychology_v2_16creatives.json

python3 $ENGINE compile $SCENE      # dry-run, see exact playlist arrays
python3 $ENGINE deploy  $SCENE      # push 4 playlists + hard-cut animation + parallel resync
python3 $ENGINE sync                # gated coordinated restart (waits for stable state)
python3 $ENGINE audit               # confirm 4 TVs on correct playlists

# When at outlet on WiFi:
python3 $ENGINE lan-check           # probe port 8000 + auth on each TV
python3 $ENGINE verify              # technical sync verification (no eyeballs needed)
python3 $ENGINE lan-sync            # tighten drift to under 30ms
