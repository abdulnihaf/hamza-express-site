Meta Ads Manager — Step by Step ›
Setting up a campaign in Meta Ads Manager:

1. Go to business.facebook.com/adsmanager
2. Click "+ Create" and select your objective (Awareness for reach, Traffic for directions)
3. Name your campaign. Turn OFF "Advantage Campaign Budget" (we control per ad set)
4. Ad Set level:
  - Budget: Lifetime Budget (required for dayparting/scheduling)
  - Schedule: Set start/end dates
  - "Run ads on schedule" toggle ON, then set time windows from your config
  - Audience: Locations, drop pin on Shivajinagar, set radius
  - Location type: "Living in or recently in"
  - Set age, languages, detailed targeting (interests) from config
5. Ad level:
  - Upload creative (image/video from AI Studio or Reels Engine)
  - Paste primary text, headline, description
  - Set CTA button + destination URL
6. Review + Publish

Key tips:
- Use Lifetime Budget (not Daily) for dayparting control
- Meta Pixel (881349531538961) auto-attaches from Ad Account settings
- For "Get Directions" CTA, use Google Maps link as destination
- Review takes 15-60 mins. Submit early!
- Ad Account ID: 634959029638544
- FB Page ID: 886791574518359
Google Ads — PMax + Search Setup ›
Performance Max (Local) Setup:

1. Go to ads.google.com (Account: 368-171-0084)
2. Click "+ New campaign", select "Sales" or "Local store visits"
3. Campaign type: Performance Max
4. Select your Google Business Profile (Hamza Express, Shivajinagar)
5. Set daily budget (minimum recommended: &rupee;1,500/day for PMax)
6. Asset Group:
  - Final URL: your Google Maps link
  - Images: Upload logo (1:1), landscape (1200x628), square (1200x1200)
  - Headlines: 3-15 headlines, 30 chars each
  - Long Headlines: up to 5, 90 chars each
  - Descriptions: up to 4, 90 chars each
  - Business name: "Hamza Express"
7. Audience Signals: Add search themes from your config
8. Location: Set radius around Shivajinagar
9. Languages: English, Hindi, Kannada, Urdu
10. Review + Publish

Search Campaign Setup:

1. Click "+ New campaign", select "Website traffic"
2. Campaign type: Search
3. Set daily budget (minimum: &rupee;100/day for Search)
4. Ad Group:
  - Keywords: Add search themes as keywords (use phrase match or broad match)
  - Location: Radius targeting around Shivajinagar
5. Responsive Search Ad:
  - Headlines (min 3, max 15): 30 chars each
  - Descriptions (min 2, max 4): 90 chars each
  - Final URL + display path
6. Review + Publish

Key tips:
- PMax requires GBP linked. Verify in "Linked accounts"
- GA4 property: G-LR6RKRV1G5 — set up conversion tracking
- PMax needs 2-4 weeks to learn. Don't change budget in first 14 days
- Search: Start with broad match + smart bidding, then refine
- Add negative keywords: "recipe", "how to make", "ingredients"
Creative Best Practices ›
Dimensions:
- Feed: 1080x1080 (square) — works for both IG + FB feed
- Stories/Reels: 1080x1920 (9:16 vertical)
- FB Landscape: 1200x628 (1.91:1) — link ads, headers
- Google PMax: Logo 1:1, Landscape 1200x628, Square 1200x1200

Meta text limits:
- Primary text: 125 chars visible (can be longer, truncated with "...See More")
- Headline: 40 chars
- Description: 30 chars
- Image text: <20% of image area (Meta may reduce delivery otherwise)

Google text limits:
- Headline: 30 chars (3-15 headlines, Google combines them)
- Long Headline: 90 chars (1-5)
- Description: 90 chars (1-4)
- Business name: 25 chars

Ad rejection risks:
- Too much text on image
- Before/after claims
- Misleading buttons or arrows on creative
- All-caps text
- For Google: text overlays on images can hurt performance
